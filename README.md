# PuzzleHub

Mobile-first vanilla JavaScript PWA for long-term Sudoku, Kakuro, and Sumplete play.

## Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Supabase
- Vercel static hosting

## Features

- 10,000 deterministic seeded levels per game
- Daily puzzles
- Auto-save after every move
- Offline-first local persistence
- Undo / redo / notes / hints
- Progress, streaks, statistics, achievements
- Installable PWA with service worker cache
- Supabase sync when authenticated session already exists on device
